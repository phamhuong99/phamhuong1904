các kiểu sữ liệu trong js
    1. kiểu số (number) 3 --> số nguyên, 3.5--> số thực
    2. kiểu chuỗi ("hello world")
    3. kiểu boolean ( kiểu chứa 2 giá trị True|False)
    4. kiểu mảng ( dánh sách các giá trị )
    5. kiểu đối tượng
    6. kiểu giá trị null
    7. kiểu undefined ( có khai báo biến nhg không khai báo giá trị)
    Mình có cái từ khóa typeof để kiểu tra kiểu dữ liệu của biến
