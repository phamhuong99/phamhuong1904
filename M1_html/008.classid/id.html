thuộc tính id trong html

    id trong html giống như là số cmt của 1 ng
    nó sẽ duy nhất trong trang html và ko được phép trùng giữa các thẻ
    id phải viết liền và không có dâu, không có ký tự đặc biệt
    tốt nhất chỉ nên có cahcs chữ cái và chữ số
    -class được sử dụng để gom nhóm giữa các thẻ html mà có các điểm chung
    1 thẻ html thì sẽ có 1 hoặc nhiều class khác nhau
    các class thì vẫn phân tách bằng dấu cách ví dụ: "mienbac top1 thudo"
    quy tắc viết class viết liền, ko dấu, ko ký tự đặc biệt
    mục đích của thuộc tính class và id là để kết hợp với các ngôn ngữ như css và javascipt
    để thực hiện các tác vụ trên trang web
