sự hác biệt của include, include_once, require, require_once

    sự giống nhau: 4 hàm này đều dùng để nạp 1 fiel vào 1 file khác
    Sự khác nhau:
    - include (khi mà quá trình nạ fike bị lỗi thì ctrinh vẫn chjay tiếp)
    - require: khi mà quá trình nạo file bị lỗi thì ctrinh sẽ dừng chạy
    - include_once giông sinclude nhg chỉ nạp file 1 lần nga cả khi bị gọi nhiều lần
    - require_once: giống require nhg chỉ nạp file 1 lần ngay cả khi bị gọi nhiều lần
