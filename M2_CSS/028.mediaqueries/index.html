Media query trong css

    Media query là kỹ thuật xây dựng những đoạn css áp dụng cho từng loại màn hình
    có độ rộng khác nhau


    @media screen and (min-width: 600px)
    cú pháp bắt đầu bằng @media
    tiếp theo kiểu màn hình screen or print
    and: điều kiện ( tức là màn hình và một điều kiện đi kèm)
     (min-width: 600px) : được hiểu là áp dụng màn hình có kích cỡ từ 600px trở lên
    (max-width: 600px) : được hiểu là áp dụng màn hình có kích cỡ từ 600px trở xuống
